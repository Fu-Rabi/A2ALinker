# A2A Linker Brand And Trademark Policy

This file explains how the A2A Linker project identity may be referenced when
the code is used under the Apache License 2.0.

It is not a separate software license. The code license is in [LICENSE](./LICENSE).

## Official Project Identity

The upstream project identity includes:

- the name `A2A Linker`
- the domain `a2alinker.net`
- project logos, wordmarks, and other brand assets published by the project

The official upstream project is maintained by **Fu-Rabi**.

The official source for A2A Linker branding and logos is:
https://www.a2alinker.net/

## What You May Do

- Refer to A2A Linker factually to describe compatibility, origin, or inspiration.
- Say that your project is based on, forked from, or compatible with A2A Linker.
- Link to the official website and repository when describing the upstream project.

## What You Must Not Do

- Do not imply that your fork, service, package, or website is the official A2A Linker project if it is not.
- Do not use the `A2A Linker` name, logo, or domain branding in a way that is likely to confuse users about source, sponsorship, or endorsement.
- Do not present modified builds as official releases from the upstream project.

Publication of a logo or other brand asset on the official website does not by
itself grant permission to reuse that asset in a way that suggests endorsement,
affiliation, or official project status.

## Guidance For Public Forks And Redistributed Versions

If you publicly redistribute a modified version, use your own branding so users
can clearly distinguish it from the upstream project.

Examples of acceptable wording:

- "Fork of A2A Linker"
- "Based on A2A Linker"
- "Compatible with A2A Linker"

Examples that are not acceptable without permission:

- "Official A2A Linker Enterprise"
- "A2A Linker Cloud" presented as an official upstream offering
- reuse of the upstream logo or website identity in a way that suggests endorsement

## Questions

For permission requests or brand questions, use the official project website:
https://www.a2alinker.net/
